# Sharing Datasets

You can share a dataset with other users using a **shared code**.

A shared code is a short key generated by DataTyper that allows other users to connect to your dataset.

---

## How to share a dataset

1. Open the dataset you want to share
2. Go to the Collaboration or Access section
3. Copy the shared code
4. Send the code to your collaborators

Anyone with the shared code can connect to the dataset.

---

## Connecting to a shared dataset

To connect to a dataset shared by someone else:

1. Open DataTyper
2. Paste the shared code into the provided field
3. Confirm the connection

The shared dataset will appear in your dataset list.
