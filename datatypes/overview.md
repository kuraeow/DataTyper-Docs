# Datatypes

Datatypes define how data inside a dataset is structured.

A datatype represents one logical category of data that you want to reuse in your designs.

Examples of datatypes:

- Names
- Email addresses
- Job titles
- Product names
- Image URLs

Each dataset can contain any number of datatypes.

---

## Datatypes and datasets

Datatypes always belong to a dataset.

- A dataset can contain multiple datatypes
- The same datatype name can exist in different datasets
- Datatypes are isolated between datasets

Datatypes help you organize data in a predictable and reusable way.
